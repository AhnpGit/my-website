    screen

    @mixin screen($size) { ... }

    Description

    Screen helper

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    Either small, medium, large or empty value

    		String— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Usage

    .foo {
  @include screen('small');
  background-color: red;
}

    Result

    .foo {
  @media screen and (min-width: 320px) {
    background-color: red;
  }
}

    @mixin stepScale($attr, $smallVal, $mediumVal, $largeVal) { ... }

    Description

    Scale any css attributes according to breakpoints

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $attr

    All kinds of css attributes. For example: width, height...

    		String— none
    $smallVal

    Measure unit for only small screens

    		Number— none
    $mediumVal

    Measure unit for medium up screens

    		Number— none
    $largeVal

    Measure unit for large up screens

    		Number— none

    Example

    Usage

    .foo {
  @include stepScale('width', 15px)
}

    Result

  @media screen and (min-width: 320px) {
    width: 15px;
  }
